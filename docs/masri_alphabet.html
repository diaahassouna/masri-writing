The ع Problem

Franco drops ع entirely or hides it behind a numeral 3. Masri gives it a real letter: Ⲵ — the Coptic Old Ain, a direct descendant of the hieroglyph for "eye." Tier 1 applies the Backtick ` which is available on every keyboard.

Keyboards & Keypads

Coptic, Greek and Latin-extended letters aren't on any physical keyboard. Here's every real way to type them — from a universal on-screen keypad that works everywhere right now, to native compose layouts for each OS, plus the custom Masri input method currently in development.

Universal Virtual Keypad works on any device, no install

Tap letters to build text below, then copy it wherever you need it.

Windows

Fastest, no install: in Word or Outlook, type the Unicode hex code then press Alt+X to convert it into the letter. Full code table below.

Recommended for everyday typing: install WinCompose (free, open source) and load the Masri sequence file. It gives you a Compose key — hold it, then type two letters, e.g. Compose → h a → Ϩ.

macOS

Built in, no install: go to System Settings → Keyboard → Text Input → Input Sources, add Unicode Hex Input. Switch to it, hold Option and type the 4-digit hex code to get the letter.

For a real compose key (so Masri letters sit on their own Option layer, mnemonic like the Linux/Windows scheme below), build a custom layout with the free app Ukelele. It's a manual build since macOS has no plain-text compose format — happy to generate a starter layout file on request.

Linux

Best option: Linux has a native, plain-text Compose system. Download the file below, save it as ~/.XCompose, then add export GTK_IM_MODULE=xim (or restart your session) so it loads. Your Compose key is whatever you've bound in your desktop's keyboard settings (often Right Alt or CapsLock).

Same mnemonic scheme as Windows: Compose → h a → Ϩ, Compose → a i → Ⲵ, and so on — full list in the file.

iOS

iOS doesn't allow custom compose files or third-party layouts at the system level, but Text Replacement gets you 90% of the way: Settings → General → Keyboard → Text Replacement → +. Add each shortcut below once — after that, typing e.g. :3 auto-expands to Ⲵ anywhere on the phone.

For anything the shortcuts don't cover (or on a shared device), use the Universal Virtual Keypad above in Safari and copy-paste — works instantly, no setup.

Android

Gboard / most keyboards: Settings → System → Languages & input → On-screen keyboard → [your keyboard] → Dictionary → Personal dictionary. Add the same shortcuts as the iOS table — type the shortcut, tap the suggestion to insert the Masri letter.

Power users: AnySoftKeyboard (open source) supports fully custom layout add-ons if you want Masri letters on a dedicated keyboard layer.
